# File Organiser

A Python automation script that automatically sorts your 
Downloads folder into organised subfolders by file type.
No more messy Downloads folder!

---

## What it does

- Moves images → Images/
- Moves PDFs and docs → Documents/
- Moves videos → Videos/
- Moves audio → Audio/
- Moves zip files → ZipFiles/
- Moves programs → Programs/
- Saves a full history log of every file moved with timestamp

---

##  How to run

1. Download `file_organiser.py`
2. Open terminal in the same folder
3. Run this command:

python file_organiser.py

4. Check your Downloads folder — all sorted!
5. Check `history.txt` in your project folder for the full log

---

## Tech used

- Python 3
- os module — file and folder operations
- shutil module — moving files
- datetime module — timestamping the log

 What I learned building this

- How to navigate and manipulate files using Python
- How to handle real-world errors (PermissionError for open files)
- How to write and append to log files
- How to use try/except for error handling
